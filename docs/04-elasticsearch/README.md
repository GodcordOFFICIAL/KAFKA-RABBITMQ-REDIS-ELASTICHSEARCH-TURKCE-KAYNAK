Elasticsearch'ı şu alanlarda kullanabilirsiniz:

### 3. **Real-time Data**

- Live dashboard'lar
- Monitoring ve alerting
- Real-time recommendations

### 4. **Machine Learning**

- Anomaly detection
- Classification
- Forecasting
